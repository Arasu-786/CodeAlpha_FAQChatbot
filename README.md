#  AI FAQ Chatbot

An intelligent FAQ Chatbot built using Python, Streamlit, and Sentence Transformers.

## Features

* Semantic question matching
* AI-related FAQ knowledge base
* Confidence score display
* User-friendly Streamlit interface
* CSV-based FAQ management

## Technologies Used

* Python
* Streamlit
* Sentence Transformers
* Pandas
* Scikit-learn
* NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run App.py
```
